# VPN Bot

Telegram бот для управления VPN подписками через XUI панель.

## Структура проекта

```
VPN_bot/
├── main.py          # Точка входа, запуск бота
├── config.py        # Загрузка конфигурации из .env
├── database.py      # Работа с SQLite базой данных
├── xui_api.py       # API для работы с XUI панелью
├── handlers.py      # Обработчики сообщений Telegram
├── .env             # Конфиденциальные данные (не в git!)
├── .env.example     # Шаблон для .env файла
├── requirements.txt # Зависимости проекта
└── README.md        # Документация
```

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

   Или создайте файл `.env` вручную и скопируйте содержимое из `.env.example`, затем заполните своими данными.

3. Заполните `.env` файл своими данными:
   - `API_TOKEN` - токен Telegram бота от @BotFather
   - `WEBHOOK_URL` - URL для webhook
   - `XUI_API` - URL XUI панели
   - `XUI_USER` - имя пользователя XUI
   - `XUI_PASS` - пароль XUI
   - `VPN_DOMAIN` - домен или IP VPN сервера

## Запуск

```bash
python main.py
```

## Модули

### config.py
Загружает переменные окружения из `.env` файла и проверяет наличие обязательных параметров.

### database.py
Работа с SQLite базой данных:
- `init_db()` - инициализация таблицы пользователей
- `user_exists()` - проверка существования пользователя
- `save_user()` - сохранение данных пользователя

### xui_api.py
API для работы с XUI панелью:
- `get_xui_cookie()` - авторизация
- `get_users()` - получение списка пользователей
- `get_new_reality_keys()` - получение Reality ключей
- `get_free_port()` - поиск свободного порта
- `create_trial_inbound()` - создание пробной подписки

### handlers.py
Обработчики команд и кнопок Telegram бота:
- `/start` - приветствие и главное меню
- `/users` - список пользователей
- Кнопки меню

### main.py
Главный файл, который:
- Инициализирует бота и диспетчер
- Регистрирует обработчики
- Настраивает webhook
- Запускает сервер

## Безопасность

⚠️ **Важно**: Никогда не коммитьте файл `.env` в git! Он содержит конфиденциальные данные.

Файл `.env` уже добавлен в `.gitignore`.
